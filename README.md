# 🌲 Dino-Christmas-Tree

**神奇的手势控制互动圣诞树** - 一个完全在浏览器中运行的3D交互式圣诞树，通过摄像头实现手势控制！

## ✨ 特色功能

- **🤲 手势控制**：左手张开/握拳控制粒子爆炸效果，右手捏合手势弹出回忆照片
- **🎨 主题切换**：一键切换圣诞🎄与新春🧧两种主题风格
- **💫 3D粒子系统**：超过18,000个动态粒子组成的炫酷圣诞树
- **🎵 音乐可视化**：背景音乐驱动粒子律动，沉浸式视听体验
- **📷 照片回忆**：上传个人照片，随机出现在树上成为装饰
- **⚙️ 完全自定义**：实时调节文字、字体、颜色、速度等参数

## 🚀 快速开始

### 在线体验
直接访问部署地址：[Dino-christmas-tree](http://christmas.094694.xyz)

### 本地运行
1. 下载 `index.html` 文件
2. 用现代浏览器（Chrome/Edge）打开
3. 点击"开始体验"，允许摄像头权限
4. 开始你的手势互动！

## 🎮 操作指南

### 手势控制
- **左手**：张开手掌 → 增加爆炸效果 | 握拳 → 减少爆炸效果
- **右手**：食指拇指捏合 → 弹出随机照片 | 松开 → 关闭照片

### 控制面板
点击右上角 ⚙️ 按钮打开控制面板：
- **文字自定义**：修改标题、字体、大小
- **风格切换**：圣诞/新春主题，多种背景选择
- **媒体控制**：上传背景音乐、管理照片
- **参数调节**：旋转速度、树的高度、音乐灵敏度

## 🛠 技术栈

- **Three.js** - 3D渲染与粒子系统
- **MediaPipe Hands** - Google手势识别AI
- **原生Web API** - 摄像头、音频、Canvas
- **纯前端** - 单HTML文件，无需后端

## 🌐 部署

### Cloudflare Pages（推荐）
1. 访问 [Cloudflare Dashboard](https://dash.cloudflare.com/)
2. 选择 Pages → Create a project → Direct Upload
3. 上传 `index.html` 文件
4. 点击 Deploy，完成！

### GitHub Pages
1. 将文件推送到GitHub仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 main 分支作为源
4. 等待部署完成

## ⚠️ 注意事项

- 需要使用**现代浏览器**（Chrome 90+ / Edge 90+）
- 首次使用需要**允许摄像头权限**
- 部分功能需要**HTTPS环境**（建议使用HTTPS访问）
- 移动设备**建议横屏体验**效果更佳
